# الخطوط المطلوبة للمشروع

يرجى إضافة الخطوط التالية في هذا المجلد:

## الخطوط العربية - Dubai
- `Dubai-Bold.ttf` - للعناوين الرئيسية
- `Dubai-Medium.otf` - للعناوين الفرعية والنصوص المتوسطة
- `Dubai-Light.otf` - للنصوص الطويلة والفقرات

## الخطوط الإنجليزية - Bristone
- `Bristone-Bold.otf` - للعناوين الرئيسية
- `Bristone-Reg.otf` - للنصوص العادية
- `Bristone-Thin.ttf` - للنصوص الخفيفة

## ملاحظات:
- تأكد من مطابقة أسماء الملفات بالضبط
- الخطوط يجب أن تكون بصيغة .ttf أو .otf
- ستتم الإشارة إليها في globals.css
