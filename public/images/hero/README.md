# صور Hero Section

يرجى إضافة الصور التالية للقسم الرئيسي:

## الصور المطلوبة
- `abdallah-hero.jpg` أو `abdallah-hero.png` - صورة عبد الله الخضر الرئيسية
  * وضعية احترافية
  * خلفية يمكن إزالتها أو استبدالها
  * جودة عالية
  * مقاس: 800x1000px على الأقل

## ملاحظات التصميم:
- سيتم إضافة gradient خلفي من #edbf8c إلى #10302b
- سيتم إضافة border-radius: 32px
- سيتم إضافة pattern_vertical_dark كخلفية زخرفية
- سيتم إضافة shadow عميق
