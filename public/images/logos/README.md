# الشعارات المطلوبة

يرجى إضافة الشعارات التالية في هذا المجلد:

## الشعارات الكاملة (مع النص)
- `logo_white.png` - الشعار الأبيض مع النص (للخلفيات الداكنة)
- `logo_dark.png` - الشعار الداكن مع النص (للخلفيات الفاتحة)

## الأيقونات فقط (بدون نص)
- `logo_white_without_text.png` - أيقونة الشعار البيضاء فقط
- `logo_dark_without_text.png` - أيقونة الشعار الداكنة فقط

## المواصفات الموصى بها:
- صيغة: PNG مع خلفية شفافة
- الأبعاد: 200-300px للشعار الكامل، 100-150px للأيقونة
- جودة عالية للوضوح على جميع الأحجام
