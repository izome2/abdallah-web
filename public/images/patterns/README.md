# أنماط الخلفية (Patterns)

يرجى إضافة أنماط الخلفية الزخرفية التالية:

## الأنماط العمودية
- `pattern_vertical_white.png` - نمط عمودي أبيض (للخلفيات الفاتحة)
- `pattern_vertical_dark.png` - نمط عمودي داكن (للخلفيات الغامقة)

## الأنماط الأفقية
- `pattern_horizontal_white.png` - نمط أفقي أبيض (للخلفيات الفاتحة)
- `pattern_horizontal_dark.png` - نمط أفقي داكن (للخلفيات الغامقة)

## المواصفات:
- صيغة: PNG مع خلفية شفافة
- أبعاد: 500x500px أو أكبر (للتكرار)
- تصميم: أنماط هندسية أو زخرفية بسيطة
- استخدام: كخلفيات زخرفية مع opacity منخفض
